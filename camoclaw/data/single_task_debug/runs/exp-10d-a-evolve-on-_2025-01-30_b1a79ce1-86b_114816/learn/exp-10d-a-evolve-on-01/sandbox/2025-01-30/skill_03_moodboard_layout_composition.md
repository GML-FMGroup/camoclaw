# Skill: Moodboard Layout & Composition Strategy

type: strategy

## When to Use

- Creating any visual moodboard for film, music video, or commercial production
- The deliverable must be professional-grade and communicate effectively to a production team
- You need to balance visual hierarchy, white space, and information density
- The moodboard will be shared with directors, art directors, cinematographers, or production designers

## One-Line Summary

A professional moodboard must use intentional layout principles—including visual hierarchy, grid structure, and strategic white space—to make reference imagery and creative direction immediately comprehensible to production teams.

## Main Body

### 1. Layout Principles for Production Moodboards

**Visual Hierarchy:**
- Primary images (hero references) should be larger and positioned prominently
- Secondary references should support, not compete with, the primary mood
- Color palette should be distinct but integrated, not overwhelming

**Grid Structure:**
- Use a 2x2 or 3x2 grid for balanced composition
- Avoid cluttered layouts—fewer, high-quality references are better than many mediocre ones
- Maintain consistent spacing between elements

**White Space:**
- White space improves readability and professional appearance
- Leave margins around the entire board (minimum 40px)
- Space between images should be consistent (20-40px recommended)

### 2. Section Organization

A production moodboard should include these sections in logical order:

| Section | Purpose | Position |
|---------|---------|----------|
| Title/Header | Project name and aesthetic description | Top |
| Color Palette | Core colors with hex values | Prominent area |
| Primary References | Film stills, key imagery | Center/Left |
| Secondary References | Supporting visuals | Right/Bottom |
| Lighting References | Specific lighting style examples | Bottom section |
| Costume/Set References | Wardrobe and production design | Separate area |
| Notes/Annotations | Creative direction text | Adjacent to relevant images |

### 3. Image Selection Criteria

**Do Include:**
- High-resolution film stills from reference films
- Specific lighting examples (chiaroscuro, neon, natural, etc.)
- Costume details (fabric texture, silhouette, color)
- Set design and location photography
- Texture references (walls, fabrics, surfaces)

**Do NOT Include:**
- Abstract color blocks or geometric shapes (these are not actionable)
- Low-resolution or watermarked images
- Unrelated stock photography
- Clip art or illustrations unless specifically requested

### 4. Typography and Annotations

**Text Guidelines:**
- Use clean, sans-serif fonts (Helvetica, Arial, or similar)
- Title: 24-32pt, bold
- Section headers: 18-20pt, semi-bold
- Body text/annotations: 12-14pt, regular
- Always include hex codes for colors: #FF5733 format

**Annotations:**
- Briefly explain WHY each reference is included
- Connect imagery to the project brief (e.g., "Lighting approach for the bridge scene")
- Keep annotations concise but informative

### 5. Common Layout Mistakes to Avoid

1. **Overcrowding**: Too many images create visual noise
2. **Inconsistent sizing**: Random image sizes look unprofessional
3. **No clear focal point**: Viewers should immediately understand the primary aesthetic
4. **Missing context**: Images without explanation may be misinterpreted
5. **Poor color presentation**: Swatches should be labeled and show the actual color

## Key Takeaways

1. **Professional moodboards are communication tools**, not art pieces—clarity trumps creativity
2. **Every element must be actionable** by a production department
3. **Consistency in sizing and spacing** creates visual polish
4. **Annotations bridge the gap** between images and execution
5. **Reference specificity matters**—vague images lead to vague results
